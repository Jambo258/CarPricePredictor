Endpoints for the car price predictor

GET user/getallusers returns all users

** requires admin

GET user/:id finds singular user details with ID

** requires admin or user with same id

POST user/register registers new user to database
POST user/login login as user which is created on registration
PUT user/:id change user account details by ID

** requires admin or user with same id

DELETE user/:id deletes user with ID

** requires admin or user with same id

POST user/predict predicts car value based on attributes like km, hp, make, year, fuel

** requires account (user or admin)
